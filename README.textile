h1. rah_pin

p. A tiny "Textpattern":http://www.textpattern.com plugin experiment that allow pinning images to Textpattern articles. Works entirely with JavaScript.

p. The project is stored here for archiving, experimenting, forking and experimenting purposes. Some of the plugin's features are unimplemented, and some don't work as implied. It doesn't work as it on it's alone.
